# HealthPath Agent - 医疗就诊路线规划智能体

## 概述

HealthPath Agent 是一个智能医疗就诊路线规划系统，帮助用户找到最合适的医院和医生。

## 功能

- **自然语言理解**: 解析用户的医疗就诊需求
- **医院号源搜索**: 从多个数据源搜索可用的医院号源
- **多维度评分排序**: 基于距离、时间、费用、排队等多个维度评分
- **PDF文档生成**: 生成美观的就医行程单

## 使用方法

### 基本调用

```python
from main_skill import execute

result = execute(
    user_input="我想在北京找个好医院看骨科，最好这周能挂上号",
    output_format="large_font_pdf"
)
```

### 输入参数

- `user_input` (str): 用户的自然语言输入，描述医疗就诊需求
- `output_format` (str): 输出格式，可选值：
  - `"large_font_pdf"` (默认): 大字版PDF，适合老年人
  - `"pdf"`: 标准版PDF

### 输出格式

返回一个字典，包含以下字段：

```json
{
  "status": "success|error|no_results",
  "steps": {
    "intent_parsing": {...},
    "hospital_search": {...},
    "decision_ranking": {...},
    "output_generation": {...}
  },
  "final_output": {
    "success": true,
    "pdf_path": "/path/to/pdf",
    "file_size_kb": 35.2
  },
  "error": null
}
```

## 工作流程

```
用户输入 (自然语言)
    ↓
[Skill 1] 意图解析
    ↓ (输出: task_params)
[Skill 2] 号源搜索
    ↓ (输出: slots)
[Skill 3] 方案决策
    ↓ (输出: recommendations)
[Skill 4] PDF生成
    ↓
最终输出: PDF文件
```

## 支持的场景

1. **老年人就医**: 腰疼、头晕等常见病症
2. **上班族就医**: 利用周末或晚上时间就医
3. **异地就医**: 跨城市医疗需求
4. **儿童急诊**: 夜间急诊需求

## 数据来源

系统支持多个数据源，优先级如下：

1. **京医通** (官方平台) - 最可靠
2. **114挂号网** (第三方平台) - 覆盖面广
3. **模拟数据** (降级方案) - 保证可用性

## 输出特点

- 🏥 **最重要的就诊信息**: 医院、科室、医生、时间
- 🚗 **交通指南**: 推荐交通方式和时间预估
- 🎒 **出门前检查清单**: 提醒携带必要物品
- 💡 **特别提醒**: 就医前的重要提示
- 📋 **推荐方案对比**: 多个医院的详细对比
- 🗺️ **交通与距离详情**: 每个方案的具体交通信息
- 📊 **就医需求摘要**: 用户需求的总结

## 特色功能

### 老年友好设计
- 大字版PDF (28pt标题, 16pt正文)
- 温暖的色彩方案 (红色/粉色主题)
- Emoji图标辅助理解
- 简洁清晰的排版

### 多维度评分算法
```
综合评分 = 0.2×距离 + 0.3×交通时间 + 0.3×费用 + 0.2×排队时间
```

### 自动降级机制
- 数据源不可用时自动切换
- 缓存机制减少网络请求
- 始终保证可用性

## 错误处理

系统会自动处理以下情况：

- 数据源不可用 → 自动切换到备选源
- 网络请求失败 → 使用缓存数据
- 没有匹配号源 → 生成"未找到"提示的PDF
- 意图解析失败 → 返回错误信息
