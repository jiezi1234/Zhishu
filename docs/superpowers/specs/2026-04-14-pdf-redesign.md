# PDF 生成模块重构设计文档

**日期**：2026-04-14  
**涉及文件**：`skills/skill_4_output/pdf_generator.py`  
**目标**：将现有丑陋的平铺式 PDF 重构为现代卡片风、全年龄通用的美观行程单

---

## 1. 设计目标

| 维度 | 当前问题 | 目标效果 |
|------|----------|----------|
| 视觉风格 | 纯文字堆叠，无层次感 | 现代卡片风，圆角卡片分块 |
| 适用人群 | 偏老年大字版 | 全年龄通用，信息密度适中 |
| 页眉 | 无 | 简洁 Logo 行：左侧图标+项目名，右侧日期 |
| 内容 | 5 个板块平铺 | 5 个板块 + 新增就诊时间轴，卡片化布局 |

## 4. 卡片规范

**卡片样式**（统一）：
- 圆角：6pt
- 左侧蓝色实线边框：3pt 宽，颜色 `#2563EB`
- 背景色：`#EFF6FF`
- 内边距：12pt（上下），14pt（左右）
- 卡片标题：14pt 加粗，颜色 `#2563EB`
- 卡片间距：10pt

### 卡片1 — 就诊信息

Key-Value 网格布局：
- 标签列：固定 60pt 宽，12pt，颜色 `#64748B`，等宽对齐（汉字填充至2字）
- 值列：剩余宽度，13pt 加粗，颜色 `#1E293B`
- 每行用 0.5pt `#E2E8F0` 细线分隔
- 字段：医院、科室、医生（含职称）、时间、预计排队

### 卡片2 — 就诊时间轴（新增）

横向4节点时间轴：
- 节点：直径 10pt 的绿色（`#10B981`）实心圆
- 连线：蓝色（`#2563EB`）虚线，线宽 1.5pt
- 节点标签（上方）：出发 / 到院 / 挂号 / 就诊，12pt 加粗
- 节点时间（下方）：自动计算，11pt，颜色 `#64748B`
- 时间计算规则：
  - 就诊时间 = `appointment_time`（来自数据）
  - 挂号时间 = 就诊时间 - 25min
  - 到院时间 = 就诊时间 - 30min
  - 出发时间 = 就诊时间 - `total_travel_time_min` - 30min

### 卡片3 — 交通建议

单段文字段落，正文12pt，包含：
- 推荐出行方式 + 预计耗时
- 综合评分 + 推荐理由

### 卡片4 — 医院对比表

表格样式：
- 表头：背景 `#2563EB`，白色字，12pt 加粗
- 奇数行：白色；偶数行：`#F8FAFC`
- 边框：0.5pt `#E2E8F0`
- 列：排名 / 医院 / 医生 / 职称 / 时间 / 费用 / 排队 / 距离 / 评分

### 卡片5 — 出行清单 + 需求摘要（左右并排）

左右各占 50%，中间用 1pt `#E2E8F0` 竖线分隔：
- 左：出行清单，每项前置 `□` 符号，12pt
- 右：需求摘要 Key-Value，科室/症状/时段/偏好，12pt

## 6. 技术实现方案

**选定方案**：ReportLab Canvas + Platypus 混合（方案 A）

### 核心架构

```python
# 新文件结构（替换现有 pdf_generator.py）
pdf_generator.py
├── draw_rounded_rect(canvas, x, y, w, h, r, fill, stroke)  # 圆角矩形工具函数
├── draw_page_header(canvas, doc)     # 页眉绘制钩子
├── draw_page_footer(canvas, doc)     # 页脚绘制钩子
├── class CardFlowable(Flowable)      # 卡片基类
│   ├── InfoCardFlowable              # 就诊信息卡
│   ├── TimelineFlowable              # 时间轴卡
│   ├── TextCardFlowable              # 交通建议卡
│   ├── TableCardFlowable             # 医院对比表卡
│   └── TwoColumnCardFlowable         # 出行清单+需求摘要卡
└── generate_pdf_document(...)        # 主入口（保持签名不变）
```

### 关键实现细节

1. **圆角矩形**：ReportLab 无原生圆角矩形，使用 `canvas.roundRect(x, y, w, h, r)` 实现（ReportLab 3.x+ 支持）
2. **卡片高度自适应**：每个 `CardFlowable.wrap()` 方法根据内容动态计算高度
3. **时间轴坐标**：在 `TimelineFlowable.draw()` 中按卡片宽度等分4段，y坐标固定居中
4. **中文字体**：保持现有 `register_chinese_fonts()` 逻辑不变，字体名传入所有 style
5. **向后兼容**：`generate_pdf_document(recommendations, task_params, output_path, large_font)` 签名不变，`output_generator.py` 无需修改

### 字体尺寸规格

| 元素 | 标准版 | 大字版（large_font） |
|------|--------|----------------------|
| 卡片标题 | 14pt | 18pt |
| 正文 | 12pt | 15pt |
| 标签/次要 | 11pt | 13pt |
| 表格内容 | 10pt | 12pt |
| 页眉/页脚 | 11pt | 13pt |
